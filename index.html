CheemsBread.xyz


I'm not one to talk about myself, but I am one to talk about computers.

This website is hosted on an OpenBSD 7.4 VPS on Vultr using the HTTPD web server.
Here are the specs of this webserver server:

  • 1 CPU
  • 512.00 MB of RAM
  • 10GB Hard Drive
  • 1TB of Bandwidth
  • The location is New Jersey
I'm still a new user of OpenBSD; I've been using GNU/Linux for desktop and my servers full-time since the spring of 2023. Recently (at the time of writing this page), I switched my web server from Debian stable to OpenBSD. There were some issues getting a web server to work with HTTPS/SSL, but once I switched to HTTPD instead of Nginx, I had no issues with OpenBSD.
