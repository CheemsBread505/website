I'm not one to talk about myself, but I am one to talk about computers.

This website is hosted on an OpenBSD 7.3 VPS on vultr using the HTTPD web server.
I rent a virtual private server to host this website. I'm going a minimal OpenBSD server:

  • 1 CPU
  • 512.00 MB of RAM
  • 10GB Hard Drive
  • 1TB of Bandwith
  • The Location is New Jersey
I'm still a new user of OpenBSD, I've been using Gnu/Linux for the desktop use and my servers fulltime since the spring of 2023. Recently (at the time of writing this page) I switched my webserver from debian stable to OpenBSD. There was some issues getting a webserver to work with https/ssl; But once I switch to HTTPD instead of nginx I have no issues with OpenBSD.
